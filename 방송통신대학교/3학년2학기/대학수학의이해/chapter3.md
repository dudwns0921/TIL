# 대학수학의 이해

## 3강. 집합

### 01 집합

#### > 집합 표현 방법

- 원소 나열법 : 집합의 원소에 속하는 모든 원소를 나열하여 표시
- 조건 제시법 : 원소가 만족하는 조건이나 관계식으로 표현

#### > 부분집합

- 집합 B에 속한 모든 원소들이 집합 A에 속할 때
- 집합 B의 모든 원소가 A에 속하지만 A와 B가 같지 않을 때, B가 A의 진부분집합

#### > 집합에서 가장 커다란 집합 U를 전체집합 혹은 모집합이라고 함

- 어떤 원소도 포함하지 않는 집합을 공집합
- 공집합은 모든 집합의 부분집합

### 02 집합의 연산

- 합집합 : A∪B = { x∈U l x∈A 또는 x∈B }
- 교집합 : A∩B = { x∈U l x∈A 그리고 x∈B }
- 여집합 : Ac = { x∈U l x∈A 또는 x∈B }
- 차집합 : A - B = { x∈U l x∈A 그리고 x∈Bc }

### 03 집합 연산의 기본정리

- 분배법칙 A∩(B∪C)=(A∩B)∪(A∩C)
  A∪(B∩C)=(A∪B)∩(A∪C)
- 드모르간 법칙 (A∪B)c = Ac∩Bc
  (A∩B)c = Ac∪Bc
  - 여집합이 연산 기호를 포함한 모든 부분에 영향을 미친다고 생각하기

### 04 원소의 수와 고전적 확률

#### > 유한집합 A의 원소의 개수는 일반적으로 기호 n(A)로 나타냄

#### > 순서쌍과 곱집합: 임의의 실수 a,b를 짝을 지어 표현한 것을 순서쌍, 집합 A와 B의 모든 순서쌍 집합 A,B가 주어졌을 때, (a,b)의 집합을 곱집합

- AXB = {(a,b)|a∈A, b∈B}
- A=B일 경우, AXB = A^2
- n(AXB) = n(A)n(B)

#### > 고전적 확률의 의미

##### > 표본공간과 사건 : 표본공간은 통계적 실험에서 발생가능한 모든 결과 집합이며 사건은 표본공간의 부분집합

##### > 사건 A가 발생할 고전적 의미의 확률

P(A) = 사건A에속하는원소의개수/표본공간의전체원소의개수
